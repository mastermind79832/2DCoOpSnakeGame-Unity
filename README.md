# 2D CoOpSnakeGame
2 player 2d co-op snake game unity. In this game, you each player can eat food to grow in size and block off the other player. If the player's head hits anybody then the player is dead. If any player length if less than zero then the player is dead. in which case the other player wins. You are provided with powerups that can either trap the other player or let you escape from tricky situations. 

Features
- Couch Mutliplayer game.
- Easily remappable controls.
- Screen Warping for both snakes.
- Eating food increases size and score.
- Eating poison Decreases size and score.
- 3 Power-ups are available.
  - score boost: increases the score gained for an interval.
  - Speed Boost: Increases the speed for an interval.
  - Sheild: Defends against head hitting anything for an interval.
